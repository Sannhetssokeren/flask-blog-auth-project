Проект: Flask-блог с авторизацией

01. Описание
Этот проект представляет собой улучшенный веб-блог с авторизацией пользователей, созданный с использованием микрофреймворка Flask.
- Реализована регистрация и вход пользователей.
- Записи в "Дневнике программиста" привязаны к пользователю.
- Используются HTML-теги, наследование шаблонов (base.html), Jinja2, стилизация с помощью CSS.
- Данные хранятся в базе данных (SQLite) с использованием Flask-SQLAlchemy и управляются через Flask-Migrate.
- Включена таблица с прогрессом по дисциплинам.

02. Структура файлов
- `app.py`: Основной файл Flask-приложения с маршрутами, логикой авторизации и привязкой к БД.
- `models.py`: Определение моделей Users и Notes.
- `requirements.txt`: Файл с зависимостями.
- `migrations/`: Директория для файлов миграций БД (генерируется Flask-Migrate).
- `instance/`: Директория для файлов, специфичных для конкретного экземпляра приложения (например, файл базы данных `blog_notes.db`).
- `templates/`: Директория для HTML-шаблонов.
  - `home.html`: Страница приветствия.
  - `register.html`: Форма регистрации.
  - `login.html`: Форма входа.
  - `base.html`: Базовый шаблон (может быть обновлен).
  - `blog.html`: Шаблон главной страницы блога.
  - `notes.html`: Шаблон страницы дневника.
- `static/`: Директория для статических файлов.
  - `style.css`: Файл стилей.
  - `img/`: (Опционально) Директория для изображений.
- `README.md`: Описание проекта.

03. Запуск
Для запуска приложения необходим Python 3.x и установленные библиотеки из `requirements.txt`.

Установка зависимостей:
```bash
pip install -r requirements.txt
```
